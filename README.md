# Usage
<!-- usage -->
```sh-session
$ npm install -g gh-stats
$ gh-stats COMMAND
running command...
$ gh-stats (-v|--version|version)
gh-stats/0.0.0 darwin-x64 node-v12.13.1
$ gh-stats --help [COMMAND]
USAGE
  $ gh-stats COMMAND
...
```
